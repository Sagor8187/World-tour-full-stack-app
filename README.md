# 🌍 World Tour – Full Stack Tour Management System

A full-stack web application for managing and booking tour packages with complete CRUD functionality. Users can create, view, update, delete, and book tours with a smooth and responsive experience.

---

## 🚀 Features

- Create tour packages
- View all available tours
- Update tour details
- Delete tours
- Tour booking system
- Fully responsive UI
- Real-time data handling
- User-friendly interface

---

## 🛠 Tech Stack

- Next.js  
- Node.js  
- Express.js  
- MongoDB  
- REST API  
- JWT Authentication (if used)

---

## 📌 Project Purpose

This project is built to simulate a real-world tour management system where users can manage travel packages and bookings efficiently using a full-stack architecture.
